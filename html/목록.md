# 목록

말 그대로 목록을 표현하고 싶을 때 사용합니다.

## ol

ordered list의 약자로 `순서가 있는` 목록을 표현할 때 사용합니다. 자식 요소는 **무조건 li**만 사용 가능합니다.

```html
<h1>실시간 급상승 검색어</h1>
<ol>
  <li>김준형</li>
  <li>junjang</li>
  <li>다이어트</li>
  <li>스타벅스</li>
  <li>KFC</li>
</ol>
```

## ul

unordered list의 약자로 `순서가 없는` 목록을 표현할 때 사용합니다. 자식 요소는 **무조건 li**만 사용 가능합니다.

```html
<h1>채용 공고</h1>
<ul>
  <li>웹 개발자</li>
  <li>서버 개발자</li>
  <li>프론트 개발자</li>
  <li>데브옵스</li>
  <li>안드로이드 개발자</li>
  <li>IOS 개발자</li>
</ul>
```

### li

list item의 약자로 ol과 ul 내부에 항목을 표현할 때 사용합니다.
